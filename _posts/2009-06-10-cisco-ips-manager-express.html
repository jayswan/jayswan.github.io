--- layout: post title: Cisco IPS Manager Express date: '2009-06-10T07:05:00.005-06:00' author: Jay Swan tags: modified_time: '2009-12-15T10:37:54.788-07:00' blogger_id: tag:blogger.com,1999:blog-5029689981158113588.post-6846693575421009758 blogger_orig_url: http://unroutable.blogspot.com/2009/06/cisco-ips-manager-express.html --- I've been doing Cisco IDS/IPS stuff recently for the first time in a long while. If you haven't tried Cisco's new free IPS Manager Express application, check it out. It makes IDS/IPS event monitoring and management reasonably useful and almost pain-free. The interface is much more intuitive that other Cisco IDS/IPS GUI products. The only problem is that the current version supports only 5 sensors; supposedly this will increase in a future release.

Added 12/15/09:
The latest version of IME supports 10 sensors.
